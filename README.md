# python-cli-todo-list
This is a simple command line interface of to-do list. There you can show, add, note and delete tasks. 
# Simple CLI To-Do List

A simple console-based application to manage tasks, written in Python.

## Features
- [x] Add tasks
- [x] View all tasks
- [x] Mark tasks as completed
- [x] Delete tasks
- [x] Error handling (prevents crashes on invalid input)

## How to run
1. Clone the repository
2. Run the script:
   python main.py

## What I learned
- Python lists and dictionaries
- While loops and control flow
- Error handling with try/except
- Input validation
